"O paciente espera em média 15 minutos na linha para apenas verificar se há horário disponível."

06 / 11

Requisitos Funcionais (RF)

O que o sistema deve fazer para atender o usuário.

RF-01: Seleção Inteligente O sistema deve filtrar dentistas por plano de saúde selecionado e exibir apenas compatíveis.
RF-02: Navegação Temporal O sistema deve exibir calendário interativo com datas e horários disponíveis.
RF-03: Transação e Confirmação O sistema deve resumir os dados antes da confirmação e gerar comprovante PDF.
RF-04: Sincronização O sistema deve atualizar a agenda interna instantaneamente após confirmação.
